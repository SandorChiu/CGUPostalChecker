# 長庚大學自動查詢郵件：完整使用教學

本教學說明如何用 **手動載入未封裝項目** 的方式安裝與使用 Chrome 外掛。

外掛名稱：**長庚大學自動查詢郵件**  
查詢網址：<https://www4.is.cgu.edu.tw/postal/studentletter.aspx>

---

## 0. 使用前重要說明

這個外掛 **不會儲存你的學校帳號與密碼**。

它的運作方式是：

1. 你先在 Chrome 裡登入長庚大學郵件收發管理系統。
2. 外掛使用 Chrome 目前已登入的狀態查詢郵件。
3. 若登入狀態失效，外掛會提醒你重新登入。

因此，請不要把學校帳號或密碼填到外掛裡。外掛設定只需要填「收件人姓名 / 單位名稱」，例如：王小明。

---

## 1. 解壓縮 ZIP

建議把外掛固定放在下面這個路徑：

```text
C:\Program Files\Google\ChromeExtensions\CGUPostalChecker
```

### 1-1. 建立資料夾

請先建立：

```text
C:\Program Files\Google\ChromeExtensions
```

如果 Windows 要求系統管理員權限，請按「繼續」或使用系統管理員帳號操作。

如果你的電腦無法寫入 `C:\Program Files`，也可以改放到自己的文件資料夾，例如：

```text
C:\Users\Admin\Documents\ChromeExtensions\CGUPostalChecker
```

但本教學預設使用：

```text
C:\Program Files\Google\ChromeExtensions\CGUPostalChecker
```

### 1-2. 解壓縮

把 `CGUPostalChecker.zip` 解壓縮後，最後應該看到這個結構：

```text
C:\Program Files\Google\ChromeExtensions\CGUPostalChecker\
├─ manifest.json
├─ background.js
├─ content.js
├─ popup.html
├─ popup.js
├─ options.html
├─ options.js
├─ history.html
├─ history.js
├─ styles.css
├─ icons\
├─ README.md
├─ 使用教學.md
├─ 使用教學.html
└─ 快速安裝.txt
```

### 1-3. 很重要：確認 `manifest.json` 的位置

Chrome 載入外掛時，必須選到 **直接包含 `manifest.json` 的資料夾**。

正確：

```text
C:\Program Files\Google\ChromeExtensions\CGUPostalChecker\manifest.json
```

如果你解壓縮後看到雙層資料夾，例如：

```text
C:\Program Files\Google\ChromeExtensions\CGUPostalChecker\CGUPostalChecker\manifest.json
```

那代表你載入時要選裡面那一層，也就是直接包含 `manifest.json` 的資料夾。

## 2. 載入 Chrome 外掛

1. 打開 Chrome。
2. 在網址列輸入：

```text
chrome://extensions/
```

3. 打開右上角「開發人員模式」。
4. 點「載入未封裝項目」。
5. 選擇這個資料夾：

```text
C:\Program Files\Google\ChromeExtensions\CGUPostalChecker
```

6. 成功後，擴充功能清單應該會出現：

```text
長庚大學自動查詢郵件
```

7. 確認它的開關是開啟狀態。

---

## 3. 將外掛釘選到 Chrome 工具列

如果 Chrome 右上角沒有看到外掛圖示：

1. 點 Chrome 右上角的拼圖圖示。
2. 找到「長庚大學自動查詢郵件」。
3. 點圖釘，將它固定在工具列。

之後就可以直接點右上角的外掛圖示操作。

## 4. 先登入長庚郵件查詢系統

請先打開：

```text
https://www4.is.cgu.edu.tw/postal/studentletter.aspx
```

如果系統要求登入，請先完成登入。

登入成功後，畫面應該會看到「查詢郵件」頁面，包含：

- 收件人
- 郵件狀態
- 收件 / 退件日期
- 日期區間
- 查詢按鈕

外掛需要使用這個已登入狀態才能查詢。

---

## 5. 第一次設定收件人

### 5-1. 打開設定頁

有兩種方式：

方式一：

1. 點 Chrome 右上角的「長庚大學自動查詢郵件」圖示。
2. 點「設定」。

方式二：

1. 到 `chrome://extensions/`。
2. 找到「長庚大學自動查詢郵件」。
3. 點「詳細資料」。
4. 點「擴充功能選項」。

### 5-2. 新增收件人

在「收件人清單」區塊按：

```text
新增收件人
```

然後填寫：

| 欄位 | 說明 | 建議 |
|---|---|---|
| 啟用 | 是否查詢這位收件人 | 打勾 |
| 收件人姓名 / 單位名稱 | 網頁查詢時輸入的名字或單位 | 例如：王小明 |
| 收件人 ID | 選填 | 先留空 |
| 郵件狀態 | 查詢未領取、全部、已領取或退件 | 建議選「未領取」 |
| 日期類型 | 收件日期或退件日期 | 建議選「收件日期」 |
| 日期區間 | 1、3、6 個月 | 建議選「1 個月」 |

填完後按：

```text
儲存設定
```

### 5-3. 多人查詢

如果要查詢多位收件人，請重複按「新增收件人」。

外掛會依序查詢每一位已啟用的收件人。

例如：

```text
王小明 → 查詢
李小明 → 查詢
王小美 → 查詢
```

沒有啟用的收件人不會被查詢。

## 6. 排程設定

在設定頁的「查詢排程」區塊可以設定：

| 設定 | 功能 |
|---|---|
| 啟用自動監控 | 開啟或關閉整體自動查詢功能 |
| 開啟 Chrome 時自動檢查一次 | 每次 Chrome 啟動後自動查一次 |
| 固定間隔自動檢查 | 依照設定分鐘數定時查詢 |
| 查詢間隔 | 建議 15–30 分鐘以上 |
| 只在指定時段查詢 | 例如只在 08:00–18:00 查詢 |
| 週末不查詢 | 六日不自動查詢 |
| 沒有查詢頁時，自動開啟背景分頁 | 若郵件查詢頁沒有開啟，外掛會自動開一個非前景分頁 |

建議初次使用設定：

```text
☑ 啟用自動監控
☑ 開啟 Chrome 時自動檢查一次
☑ 固定間隔自動檢查
查詢間隔：30 分鐘
☐ 週末不查詢，依需求決定
☑ 沒有查詢頁時，自動開啟背景分頁
```

## 7. 通知設定

在「通知設定」區塊可以設定：

| 設定 | 建議 | 說明 |
|---|---|---|
| 只有查到郵件才跳通知 | 開啟 | 沒有郵件時只寫入紀錄，不干擾使用者 |
| 同一天同一收件人只提醒一次 | 開啟 | 今天提醒過就不再一直跳通知 |
| 登入失效或找不到查詢頁時提醒 | 開啟 | 避免外掛失效但使用者不知道 |
| 在外掛圖示顯示目前查到的郵件總數 | 開啟 | 右上角 icon 顯示總件數 |

### 7-1. 今日提醒一次的邏輯

如果開啟「同一天同一收件人只提醒一次」：

| 狀況 | 會不會跳通知 |
|---|---|
| 今天第一次查到王小明有郵件 | 會 |
| 今天第二次又查到王小明仍有郵件 | 不會，只寫入紀錄 |
| 今天查到李小明也有郵件 | 會，因為是不同收件人 |
| 明天王小明郵件還沒領 | 會再提醒一次 |
| 沒有郵件 | 不會，只寫入紀錄 |

如果要重新測試通知，可以按：

```text
重置今日提醒紀錄
```

## 8. 立即查詢一次

設定完成後，建議先手動測試：

1. 先確認長庚郵件查詢頁已登入。
2. 點 Chrome 右上角外掛圖示。
3. 按「立即查詢一次」。

外掛會依序查詢所有已啟用的收件人。

查詢完成後：

- 有新郵件：跳 Chrome 通知。
- 沒有新郵件：只寫入背景紀錄，不跳通知。
- 登入失效：跳通知提醒重新登入。

## 9. 右上角 icon 數字與滑鼠提示

### 9-1. icon 數字 badge

右上角外掛 icon 的數字會依照 **各收件人自己設定的郵件狀態** 加總。

例如：

| 收件人 | 郵件狀態 | 查到件數 |
|---|---|---:|
| 王大明 | 未領取 | 0 |
| 陳小明 | 全部 | 3 |

外掛 icon 會顯示：

```text
3
```

### 9-2. 滑鼠移到 icon 的提示文字

把滑鼠游標移到外掛 icon 上，會顯示類似：

```text
長庚大學自動查詢郵件
王大明  未領取  0件
陳小明  全部  3件
```

如果尚未查詢過，會先顯示 `0件`；第一次查詢完成後會更新。

點外掛圖示後按：

```text
紀錄
```

可以查看最近查詢紀錄，例如：

| 時間 | 類型 | 收件人 | 數量 | 訊息 |
|---|---|---|---:|---|
| 2026/6/12 09:30 | 查到郵件 | 王小明 | 2 | 查到 2 筆，新增 1 筆 |
| 2026/6/12 10:00 | 今日已提醒 | 王小明 | 2 | 今天已提醒過，因此本次不再跳通知 |
| 2026/6/12 10:30 | 無郵件 | 李小明 | 0 | 沒有查到郵件 |

紀錄頁也可以匯出 JSON，以便之後除錯或備份。

## 12. 更新外掛

如果之後取得新版 ZIP，請照這樣更新：

1. 關閉或停止外掛監控。
2. 解壓縮新版 `CGUPostalChecker.zip`。
3. 覆蓋原本資料夾：

```text
C:\Program Files\Google\ChromeExtensions\CGUPostalChecker
```

4. 打開 Chrome：

```text
chrome://extensions/
```

5. 找到「長庚大學自動查詢郵件」。
6. 按「重新載入」圖示。
7. 按「立即查詢一次」測試。

一般情況下，外掛設定會保留在 Chrome 的擴充功能儲存空間中，不會因覆蓋檔案而消失。

## 13. 常見問題排除

### Q1. 按「立即查詢一次」沒有反應

請確認：

1. 長庚郵件查詢頁是否已登入。
2. 設定頁是否至少有一位啟用中的收件人。
3. `chrome://extensions/` 裡外掛是否有紅色錯誤訊息。
4. 是否勾選了「只在指定時段查詢」，但目前不在允許時間內。

### Q2. 一直提醒要重新登入

可能是長庚系統的登入 session 已過期。

請重新打開：

```text
https://www4.is.cgu.edu.tw/postal/studentletter.aspx
```

完成登入後，再按「立即查詢一次」。

### Q3. 收件人查不到資料

可能原因：

1. 收件人姓名輸入與網站 autocomplete 不一致。
2. 網站需要選到 autocomplete 的正式收件人項目。
3. 收件人 ID 需要填入。

建議先在原網站手動輸入同一個名字測試。若手動可以查到，但外掛查不到，再回報查詢結果頁面的 HTML 以便調整解析。

### Q4. 為什麼沒有郵件時不跳通知？

這是設計行為，避免干擾使用者。沒有郵件或沒有新郵件時，只會寫入查詢紀錄。

### Q5. 為什麼今天只提醒一次？

如果開啟「同一天同一收件人只提醒一次」，外掛會記錄今天已提醒過的收件人。今天再次查到同一人的郵件時，只會寫入紀錄，不再跳通知。

### Q6. 右上角 icon 沒有數字

可能原因：

1. 尚未查詢過。
2. 查到的總件數是 0。
3. 通知設定裡「在外掛圖示顯示目前查到的郵件總數」沒有勾選。

### Q7. 滑鼠移到 icon 沒有顯示每個人的件數

請先按一次「立即查詢一次」。第一次查詢完成後，tooltip 會更新成各收件人的狀態與件數。

## 15. 隱私與安全

- 外掛不儲存學校帳號密碼。
- 外掛只在 `https://www4.is.cgu.edu.tw/postal/studentletter.aspx` 查詢頁運作。
- 收件人設定、查詢結果數量、查詢紀錄會存在 Chrome 本機擴充功能儲存空間。
- 若要移除所有資料，可在 Chrome 移除此外掛，或在設定頁清除已看過紀錄與查詢紀錄。
